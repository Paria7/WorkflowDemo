npm # @bci-ng-material-theme

Bosch corporate design theme for angular material.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory. 

Run `yarn dev-build` to build the project and automatically push it into the demo app.

## Using local dependencies
1. run `yarn link` in the dist-folder of the dependency
2. run `yarn link <dependency>` in this project to link it
